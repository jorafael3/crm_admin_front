Use the cdkDropList directive to create a grid of sortable items that can be rearranged by dragging and dropping. The cdkDropListData binds the data (like randomSortable) to the grid, allowing the items to be dynamically reordered. The cdkDropListOrientation is set to "mixed" to enable both horizontal and vertical dragging. Each item in the grid is made draggable with the cdkDrag directive. This example demonstrates how to use a grid layout where the items can be shuffled randomly by dragging, providing an interactive and flexible sorting experience.
