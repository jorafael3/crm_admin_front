Use the cdkDropList directive to create a sortable list with drag handles. The cdkDragHandle directive is applied to the handle icon, allowing the user to click and drag items to reorder them within the list. The cdkDropListData binds the data (like sortableList) to the drop list, while the cdkDropListDropped event handler triggers the drop() method to update the item order after a drop. This example demonstrates a "handle-based" drag-and-drop system where only the specified handle icon can be used to drag and reorder items, providing more control over the list's interactivity.
