Use the cdkDropList directive to create a draggable list, but with the sorting disabled. The cdkDropListDisabled property is set to true, which prevents any dragging and sorting actions from occurring within the list. The cdkDropListData binds the data (like disableList) to the list, but since sorting is disabled, the list remains static.
