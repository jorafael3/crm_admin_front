Use the cdkDropList directive to create a basic draggable list, where items can be reordered by dragging and dropping. The cdkDrag directive is applied to individual list items, making them draggable within the container. The cdkDropListDropped event handler, triggered by the drop() method, updates the order of items when they are dropped in a new position.
