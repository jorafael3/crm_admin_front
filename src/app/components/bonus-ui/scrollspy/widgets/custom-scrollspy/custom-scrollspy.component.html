To implement a Custom Scrollbar with Scrollspy, add a div inside the card body with the classes custom-scrollspy-section and nested-scrollspy. This ensures smooth scrolling within a defined area while tracking active sections. Apply ngbScrollSpyMenu to the navigation and use ngbScrollSpyItem for each item to link it to its corresponding content section. Inside the scrollable content area, mark sections with ngbScrollSpyFragment to enable automatic highlighting. Scroll through the sections below to see the active class update dynamically.
